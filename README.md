# TIY Blog Clone

This application uses pure CSS to recreate The Iron Yard's home website. Enjoy!


## Screenshot
![TIY Blog Clone](/dist/assets/images/blog.png)

## Installation

```
git clone https://github.com/mkhira2/tiyBlogCloneReact.git
cd tiyBlogCloneReact
npm install
npm run go

open localhost:3000
```
